## 2. ¿Qué es un sitio web?

Antes de aprender a crear páginas web, primero debemos entender qué son.

Una página web es un conjunto de información que podemos ver a través de Internet utilizando un navegador como Chrome, Edge o Firefox.

Por ejemplo:

* Redes sociales
* Tiendas virtuales
* Plataformas educativas
* Blogs
* Sitios de noticias

Todos ellos son páginas o sitios web, pero detrás de cada página web existe código.

Uno de los lenguajes más utilizados para crear ese código es HTML.

Los navegadores leen e interpretan ese código y luego muestran el resultado visual que nosotros vemos en la pantalla.

Es parecido a una receta de cocina donde:

* El código sería la receta.
* El navegador sería el cocinero.
* La página que vemos sería el plato terminado.

## 3. ¿Cómo funciona una página web?

Veamos qué ocurre cuando visitamos una página web.

### Lo que hace el usuario

1. Abre el navegador.
2. Quiere visitar una página.
3. Escribe una dirección web.
4. Presiona Enter.
5. La página aparece en pantalla.

Parece algo muy sencillo pero detrás de escena suceden más cosas.

### Lo que ocurre en Internet

1. El usuario escribe una dirección web.
2. El navegador envía una solicitud.
3. Un servidor recibe la solicitud.
4. El servidor responde enviando archivos.
5. El navegador recibe esos archivos.
6. Finalmente muestra la página web.

Gracias a este proceso podemos acceder a cualquier página web en cuestión de segundos.

## 4. Tecnologías esenciales para el desarrollo web

Para construir una página web normalmente utilizamos varias tecnologías.

### HTML

HTML se encarga de estructurar el contenido.

Con HTML podemos crear:

* Títulos
* Párrafos
* Imágenes
* Enlaces
* Listas

Podemos imaginar HTML como el esqueleto de una página web. Sin él, la página no tendría estructura.

### CSS

CSS se encarga de la apariencia visual.

Con CSS podemos modificar:

* Colores
* Tamaños
* Posiciones
* Diseños
* Animaciones sencillas

Si HTML es el esqueleto, CSS sería la ropa, los colores y la decoración.

### JavaScript y TypeScript

Estas tecnologías permiten que una página sea interactiva. Gracias a ellas podemos:

* Mostrar mensajes
* Validar formularios
* Crear menús interactivos
* Actualizar información sin recargar la página

JavaScript es el lenguaje más utilizado para agregar interactividad.

TypeScript es una versión que añade herramientas adicionales para ayudar a los programadores a detectar errores más fácilmente.

Ambos son completamente válidos para desarrollar páginas web modernas.

Sin embargo, en este curso nos enfocaremos principalmente en HTML y CSS para construir una base sólida.

## 5. Frontend

Ahora hablemos de una palabra muy común en el desarrollo web: Frontend.

La palabra Front significa "frente" o "fachada".

El Frontend es toda la parte de la página que el usuario puede ver e interactuar. Por ejemplo:

* Textos
* Imágenes
* Botones
* Menús
* Colores
* Diseño visual

Todo lo que aparece en pantalla forma parte del Frontend.

## 6. Backend

También existe otra parte llamada Backend. La palabra Back significa "atrás".

El Backend es la parte interna que normalmente el usuario no ve.

Este se encarga de:

* Almacenar datos
* Gestionar usuarios
* Ejecutar reglas y procesos
* Comunicarse con servidores
* Procesar información

Por ejemplo, cuando inicias sesión en una red social, el Backend verifica si tu usuario y contraseña son correctos.

## 7. Frontend + Backend

Las páginas web modernas funcionan gracias a la colaboración entre Frontend y Backend.

Podemos imaginarlo como un restaurante:

* El Frontend sería el comedor que ven los clientes.
* El Backend sería la cocina donde se prepara todo.

Si ambos funcionan correctamente, la experiencia del usuario será satisfactoria.
